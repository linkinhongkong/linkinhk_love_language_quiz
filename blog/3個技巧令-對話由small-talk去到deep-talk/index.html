3個技巧令 對話由small talk去到deep talk

唔係你悶,係你未用啱方法 👉

戀愛溝通學

💬 Small Talk 陷阱

好多人識新朋友或者第一次約會,成晚都係講天氣、返工、放假去邊。表面上好有禮貌,但其實兩個人都覺得悶,冇真正了解對方。點解會咁?因為大家都驚問得太深會唐突,所以停留喺 safe topics。但心理學研究發現:Deep talk 先會建立真正嘅 connection。

技巧 1 | 用 Follow-up Questions 🔍

錯誤示範:
你:「你做咩工?」
佢:「做老師」
你:「咁好呀!」
[話題完]

正確示範:
你:「你做咩工?」
佢:「做老師」
你:「教咩科?點解會揀教書?」
佢:「教中文。因為我鍾意睇學生成長」
你:「聽落你好有熱誠!有咩 moment 令你覺得最有意義?」

關鍵:每次回應都 dig deeper
由事實 → 原因 → 感受
先會真正了解一個人

技巧 2 | 分享你嘅感受 💛

錯誤示範:
你:「你做咩工?」
佢:「做老師」
你:「咁好呀!」
[話題完]

正確示範:
你:「你做咩工?」
佢:「做老師」
你:「教咩科?點解會揀教書?」
佢:「教中文。因為我鍾意睇學生成長」
你:「聽落你好有熱誠!有咩 moment 令你覺得最有意義?」

關鍵:每次回應都 dig deeper
由事實 → 原因 → 感受
先會真正了解一個人

技巧 3 | 問開放式問題 ❓

錯誤示範 (封閉式):
「你鍾唔鍾意你份工?」→「鍾意」
「你有冇興趣?」→「有」
「你 weekend 去咗邊?」→「屋企」
[完全接唔落去]

正確示範 (開放式):
「你份工最 enjoy 邊部分?」
「你平時鍾意做咩?點解會鍾意?」
「上個 weekend 你做咗咩令你開心嘅事?」

關鍵:問「點解」「點樣」「咩感覺」
迫佢諗深啲、講多啲
自然會去到 deep talk

實戰 Checklist 💡

✅ 準備 3-5 條 deep questions
例如:「你最近有咩 challenge?」「你覺得咩對你最重要?」

✅ 用 70/30 rule
70% 時間 listen,30% 時間講。Deep talk 係關於了解對方,唔係 show off

✅ 留意對方反應
如果佢答得好簡短,可能未準備好 deep talk。慢慢嚟,唔好 push

❌ 唔好問審問式問題
問完「做咩」就問「人工幾多」「幾時買樓」→ 呢啲係 interview 唔係傾偈

記住:Deep talk 係雙向,唔係單向 interrogation

技巧 + 啱嘅對象 = 真正嘅 connection 💛

你可以學晒呢啲技巧,但如果對方:
❌ 唔想 deep talk,淨係想 small talk
❌ 唔願意分享真實感受
❌ 答嘢好 defensive

你再 try hard 都冇用。
